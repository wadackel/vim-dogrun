# dogrun generator

> A tool that generates pre-optimized colorscheme.

## Prerequisites

Install required development tools using mise:

```shell
$ mise install
```

## Development

### Output files

```shell
$ just build
```

### Watch

```shell
$ just watch
```

### Output stdout

```shell
$ just stdout
# or
$ just debug
```

### Run tests

```shell
$ just test
```

### Format code

```shell
$ just fmt
```

### Check all (lint with format check, test)

```shell
$ just check
```

### Build release binary

```shell
$ just release
```
